
#### 关卡设计：`level-design.md`

```markdown
---
name: level-design
version: 1.0.0
description: "关卡结构与体验设计技能，生成关卡数据和节奏曲线"
---

## 功能说明
基于核心循环和难度曲线，为游戏生成关卡结构、目标体验和对应配置数据。

## 输入参数
- 游戏类型（Roguelike / 关卡制 RPG / 塔防等）
- 计划关卡数量（如 10 关）
- 难度趋势（递增 / 波动 / 先易后难）

## 输出格式

1. 关卡概要表（Markdown 表格）

| 关卡ID | 名称       | 目标体验       | 引入机制       | 目标难度 |
|--------|------------|----------------|----------------|----------|
| 1      | 新手教学关 | 学习基础操作   | 基础战斗       | 0.4      |
| 2      | 进阶试炼   | 小幅挑战提升   | 新敌人类型     | 0.5      |

2. 可落地的 JSON 配置示例：

```json
{
  "levels": [
    {
      "id": 1,
      "name": "新手教学关",
      "description": "基础机制教学",
      "target_difficulty": 0.4,
      "content": "简单敌人、基础资源"
    },
    {
      "id": 2,
      "name": "第二关",
      "description": "难度提升",
      "target_difficulty": 0.5,
      "content": "增加机制复杂度"
    }
  ]
}

已有核心循环和难度曲线，请用 level-design 技能生成

# 主技能文件：game_design_from_book

## 目标
将一本游戏设计书/手册转化为 Claude Code 可执行的 8 阶段游戏设计流程，用于从 0 到 1 设计并实现一款游戏。

## 八阶段流程
1. 【概念定义】分析游戏愿景、核心命题和目标玩家
2. 【核心循环】拆解游戏核心循环，输出 mermaid 流程图
3. 【机制与平衡】定义数值公式、难度调整机制
4. 【关卡与体验】设计关卡结构、节奏曲线
5. 【视觉与交互】规划 UI 布局和交互反馈
6. 【玩家心理】分析玩家动机和挫折点
7. 【原型与测试】制定 MVP 测试计划与迭代策略
8. 【团队管理】任务拆解与迭代规划（可选）

## 使用说明

### 触发方式
- 当用户提到「游戏设计」「关卡设计」「数值平衡」「从 0 做一款游戏」等需求时，应优先考虑使用本技能。
- 在 Claude Code 项目中，用户可以显式发起：
  - “请按 game_design_from_book 的 8 阶段流程，帮我从 0 设计这款游戏。”

# Quality Assurance Standards

## Quality Assurance Planning
- **IEEE Std 730-2002**: Standard for Software Quality Assurance Plans

## Reviews
- **IEEE Std 1028-1997**: Standard for Software Reviews
  - Covers management reviews, technical reviews, inspections, walkthroughs, and audits

## Testing
- **IEEE Std 1008-1987 (R1993)**: Standard for Software Unit Testing
- **IEEE Std 829-1998**: Standard for Software Test Documentation
  - Covers test plans, test design specifications, test case specifications, test procedures, test item transmittal reports, test logs, test incident reports, and test summary reports

## Quality Metrics
- **IEEE Std 1061-1998**: Standard for Software Quality Metric Methodology

## Usage Notes
- IEEE 730 provides the overall QA framework
- IEEE 1028 defines review types and procedures
- IEEE 829 templates can be adapted for different testing methodologies
- IEEE 1061 helps establish measurable quality criteria

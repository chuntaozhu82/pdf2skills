# Software Engineering Standards Lists

## Software Development Standards

Use these standards when working on development activities:

| Standard | Title | Purpose |
|----------|-------|---------|
| IEEE Std 830-1998 | Recommended Practice for Software Requirements Specifications | Software requirements specification |
| IEEE Std 1233-1998 | Guide for Developing System Requirements Specifications | System requirements specification development |
| IEEE Std 1016-1998 | Recommended Practice for Software Design Descriptions | Software design description |
| IEEE Std 828-1998 | Standard for Software Configuration Management Plans | Software configuration management planning |
| IEEE Std 1063-2001 | Standard for Software User Documentation | Software user documentation |
| IEEE Std 1219-1998 | Standard for Software Maintenance | Software maintenance |

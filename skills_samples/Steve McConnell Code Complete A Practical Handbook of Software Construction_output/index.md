# Code Complete Skills Collection

This collection of skills is derived from Steve McConnell's seminal work, "Code Complete: A Practical Handbook of Software Construction." It provides domain-specific guidance to help users write higher-quality, maintainable, and robust software. These skills cover a wide range of topics, including code construction, layout, commenting strategies, design methodologies, and professional development practices.

## How to Use

To leverage this collection, you can either:
1.  **Direct Invocation**: Explicitly ask to use a specific skill by name (e.g., *"Apply the 'pseudocode-programming-process' skill to this algorithm."*)
2.  **Contextual Routing**: Describe your current task or problem (e.g., *"I need to refactor this deeply nested if-statement"*), and Claude will identify and apply the most relevant skill (e.g., *complex-conditional-refactoring*).

These skills are designed to be integrated into your daily workflow, providing checklists, strategies, and principles from "Code Complete" to elevate the quality of your software construction.

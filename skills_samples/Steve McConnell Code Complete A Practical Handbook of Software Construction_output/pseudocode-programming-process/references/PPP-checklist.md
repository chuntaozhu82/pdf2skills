# PPP Checklist

## Preparation and Design

- [ ] Preconditions confirmed and documented
- [ ] Problem clearly defined
- [ ] Routine designed with error handling
- [ ] Performance considerations addressed
- [ ] Data structures designed
- [ ] Available libraries identified
- [ ] Routine named descriptively

## Pseudocode

- [ ] Algorithms researched if needed
- [ ] Pseudocode steps written
- [ ] High-level comments added
- [ ] Header comments complete (purpose, parameters, returns)

## Implementation

- [ ] Code written below corresponding comments
- [ ] Pseudocode translated accurately
- [ ] Declarations generated correctly

## Verification

- [ ] Code checked for errors
- [ ] Code stepped through mentally or with debugger
- [ ] All errors removed
- [ ] Refactoring completed
- [ ] Code cleaned up
- [ ] Pseudocode reviewed for consistency
- [ ] Tests written and passed
- [ ] Iteration completed until quality achieved

## Final Quality

- [ ] Routine meets all requirements
- [ ] Error handling is comprehensive
- [ ] Code is well-documented
- [ ] Code is readable and maintainable
- [ ] Performance is acceptable
